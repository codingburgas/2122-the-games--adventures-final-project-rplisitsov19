About Trapped Developer

Trapped developer is a story progression, turn based text game which is inspired by the popular game Metal Gear Rising: Vengeance. Trapped developer allows the user to attack enemies using attack commands and change weapons. Despite it being relatevily short, the game has a great storyline which you can follow along. You also get to choose which path you take, but be careful! Some enemies are stronger than others.

The Developer starts out with a stick and gradually upgrades his weapons as the game progresses.

The Wolf is one of the many opponents you will face in this game. He may seem easy to defeat, but you shouldn't underestimate his abilities.
